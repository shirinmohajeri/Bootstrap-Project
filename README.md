## 📚 Important Bootstrap 5 Changes

Bootstrap 5 introduced new syntax changes:

| Bootstrap 4 | Bootstrap 5 |
|---|---|
| `data-toggle` | `data-bs-toggle` |
| `float-right` | `float-end` |
| `btn-block` | `w-100` |
